# Stories Viewer

An Instagram-like stories viewer built with React and TypeScript.

## Features

- Horizontal scrollable story avatars
- Auto-advancing stories with progress bars
- Tap left/right to navigate between stories
- Viewed stories marked with gray ring
- Mobile-responsive design

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

React 19, TypeScript, Tailwind CSS 4, Vite
